# DevConnect

A social platform for developers to share projects, write technical articles, and connect with others.

## Tech Stack

- Next.js 14 with App Router
- PostgreSQL with Prisma ORM
- NextAuth.js for authentication
- Tailwind CSS for styling
- TypeScript

## Features

- Developer profiles with skills and bio
- Project showcases with live demos
- Technical blog with Markdown support
- Follow system and activity feed
- Search developers by skills

## Getting Started

```bash
git clone https://github.com/hannan-rizwan/DevConnect.git
cd DevConnect
npm install
cp .env.example .env
npx prisma db push
npm run dev
```
